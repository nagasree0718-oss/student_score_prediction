# Student Score Prediction

## Project Overview
This machine learning project predicts student exam scores based on study habits, attendance, sleep hours, previous grades, and other factors. It uses feature engineering, scaling, and a stacked ensemble of Random Forest and Gradient Boosting/XGBoost to achieve high prediction accuracy.

---

## Features Used
- Hours Studied
- Attendance (%)
- Sleep Hours
- Previous Scores
- Parental Involvement
- Access to Resources
- Extracurricular Activities
- Motivation Level
- Internet Access
- Tutoring Sessions
- Family Income
- Teacher Quality
- School Type
- Peer Influence
- Physical Activity
- Learning Disabilities
- Parental Education Level
- Distance from Home
- Gender

---

## Model
- Stacked Ensemble Regressor:
  - Random Forest Regressor
  - Gradient Boosting Regressor
  - Optional: XGBoost Regressor
- Features scaled using StandardScaler
- Trained to predict continuous exam scores (Regression problem)
